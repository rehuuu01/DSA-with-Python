DAY 51 — Count Subsets With Sum K

This is another variation of the 0/1 Knapsack pattern.

Goal:
Count how many subsets form a given sum.

State:
dp[j] = number of ways to form sum j

Transition:
dp[j] += dp[j - num]

Important:
Loop backwards to avoid using the same element multiple times.

Base Case:
dp[0] = 1

Meaning:
There is exactly one way to form sum 0 (empty subset).

Pattern Reminder:
This is the same core DP used in:

- Subset Sum
- Equal Partition
- Target Sum
- Partition With Given Difference
