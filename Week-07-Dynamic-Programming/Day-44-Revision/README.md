# Day 44 - DP Revision (0/1 Knapsack Pattern Reinforcement)

## Topics Revised
- 0/1 Knapsack Core Pattern
- Subset Sum
- Equal Partition
- Count of Subsets with Given Sum
- Target Sum (Subset Transformation)

## Key Learning
All these problems follow the same 0/1 Knapsack template.
The only difference is:
- Boolean DP (True/False)
- Count DP (Number of ways)
- Mathematical reduction (Target Sum)

## Pattern Recognition
If problem says:
- "Choose or Not Choose"
- "Each element used at most once"
- "Subset related"

Think → 0/1 Knapsack Pattern.
