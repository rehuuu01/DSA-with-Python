1Ô∏è‚É£ WHAT IS DIJKSTRA?

Dijkstra is a greedy algorithm that finds the shortest path
from a source node to all other nodes in a weighted graph
with NON-NEGATIVE edge weights.

---

2Ô∏è‚É£ MAIN IDEA

Always expand the node with the smallest distance first.

This is achieved using:
‚Üí Min Heap (Priority Queue)

---

3Ô∏è‚É£ EDGE RELAXATION

If:

distance[u] + weight < distance[v]

Then update distance[v]

This process is called RELAXATION.

---

4Ô∏è‚É£ WHY SKIP OUTDATED ENTRIES?

When we push updated distances into the heap,
older larger values may still exist inside the heap.

So we check:

if curr_dist > dist[node]:
    continue

This avoids extra work.

---

5Ô∏è‚É£ BFS VS DIJKSTRA

BFS:
- Used for unweighted graphs
- Uses Queue
- Time: O(V + E)

Dijkstra:
- Used for weighted graphs
- Uses Min Heap
- Time: O((V + E) log V)

---

6Ô∏è‚É£ LIMITATION

Does NOT work for negative weights.

For negative weights:
‚Üí Use Bellman-Ford Algorithm

---
